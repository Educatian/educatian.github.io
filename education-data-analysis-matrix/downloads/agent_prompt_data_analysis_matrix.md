# Agent Prompt - Data Analysis Matrix Audit

You are helping me map an educational dataset to defensible analysis options.

Available data:

```text
<paste dataset description, files, variables, IDs, timestamps, text/video/sensor sources, intervention design, outcomes, sample/context>
```

Candidate goal:

```text
<what I hope to learn or publish>
```

Create a matrix with these columns:

1. Data type
2. Raw forms available
3. Unit of analysis
4. What the data can show
5. Analysis families that fit
6. Claim verbs supported
7. Common overclaim to avoid
8. Minimum validity checks
9. Example paper genre
10. Extra evidence needed for stronger claims

Rules:

- Do not invent missing data.
- Separate observed behavior from inferred constructs.
- Flag causal, measurement, prediction, and generalization claims separately.
- If a research question is not supported, say so directly and suggest safer wording.
